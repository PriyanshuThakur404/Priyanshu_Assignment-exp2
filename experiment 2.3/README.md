## 1. Aim

To build an interactive SVG drawing tool with mouse event handlers.

## 2. Objectives

- Create an SVG canvas component
- Implement pointer/mouse event tracking
- Add shape drawing (pen, line, rectangle, circle)
- Include color selection, undo, clear, and download features
- Make the canvas responsive using viewport units and SVG viewBox

## 3. Hardware / Software Requirements

- **Hardware:** i5 CPU, 8GB RAM
- **Software:** Chrome or Firefox (SVG-compatible)

## How to run

1. Open `experiment 2.3/index.html` in a modern browser.
2. Use the toolbar to choose tools and colors. Use <kbd>U</kbd> to undo.
