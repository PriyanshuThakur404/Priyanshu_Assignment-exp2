# Experiment 2.1 — Live Word Counter

## 1. Aim

To implement a simple live word counter that enforces a 150-word input limit.

## 2. Objectives

- Provide a clean, responsive text input
- Show live word count and remaining words
- Prevent typing past 150 words
- Keep styling minimal and accessible

## 3. Hardware / Software Requirements

- **Software:** VS Code, React 18+, Chrome DevTools
- **Hardware:** i3+ CPU, 4GB RAM

## 4. About the Program

This experiment demonstrates reactive UI updates using React hooks for a simple, accessible word-limited input.

## How to run ✅

1. Open `experiment 2.1/index.html` in a modern browser (Chrome/Edge/Firefox).
2. For a local server (recommended), run `npx serve` or `python -m http.server` in the `experiment 2.1` folder and open `http://localhost:5000` (or the port shown).

## Features

- Enforces a 150-word limit
- Live remaining words indicator
- Accessible announcements via `aria-live`
